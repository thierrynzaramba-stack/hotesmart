1. Démarrer

Le calendrier est le cœur de la gestion de vos biens. Chaque modification que vous y faites (prix, disponibilité, durée minimale de séjour…) est enregistrée puis publiée automatiquement sur vos plateformes de réservation connectées.

Pour ouvrir le calendrier d'un bien : rendez-vous dans Mes biens, puis cliquez sur Calendrier sur le bien souhaité.

Une date pour laquelle vous n'avez rien configuré utilise automatiquement le prix de base du bien. Vous ne configurez que les exceptions (week-ends, vacances, périodes spéciales…).

2. Les deux interfaces

Sur ordinateur Ordinateur

Le calendrier s'affiche sous forme de tableau horizontal : les jours en colonnes, les informations en lignes (Prix, Disponibilité, etc.). Vous pouvez afficher plusieurs biens en même temps, empilés, pour comparer.

L'édition se fait comme dans un tableur : cliquez sur une case, tapez une valeur, validez avec Entrée. Vous pouvez sélectionner plusieurs cases d'un coup pour appliquer la même valeur.

Sur mobile Mobile

Le calendrier s'affiche sous forme de calendrier mensuel classique, un bien à la fois. Naviguez de mois en mois avec les flèches. En haut, deux menus : le bien et la vue (ce que les cases affichent : tarifs, durée minimale, disponibilité…).

Touchez une date pour ouvrir le panneau de configuration. Vous pouvez installer l'application sur votre écran d'accueil pour un accès rapide.

📲 Installer l'application sur iPhone
  1. Ouvrez le calendrier dans Safari
  2. Touchez le bouton Partager
  3. Choisissez « Sur l'écran d'accueil »

3. Toutes les options expliquées

Voici chaque réglage que vous pouvez appliquer à une date ou une période, et quand l'utiliser.

Prix de base

Le tarif d'une nuit. Utilisez-le pour ajuster vos prix selon les saisons, week-ends, événements locaux.

Prix par personne

Définit le nombre de voyageurs inclus dans le prix de base, et un supplément par voyageur additionnel. Utile si vous facturez plus cher au-delà d'un certain nombre d'occupants.

Disponibilité

Ouvre ou ferme une date à la réservation. Une date fermée apparaît en rouge clair. Fermez les dates où vous ne souhaitez pas louer (usage personnel, travaux…).

Minimum de nuits à l'arrivée

Nombre minimal de nuits qu'un voyageur doit réserver s'il arrive ce jour-là. Classique pour imposer 2 ou 3 nuits le week-end, ou une semaine en haute saison.

Minimum de nuits sur le séjour

Durée minimale imposée si le séjour traverse cette date. Pour protéger des périodes entières (vacances scolaires par exemple).

Maximum de nuits

Durée maximale d'un séjour incluant cette date. Pour éviter les très longs séjours sur des périodes prisées.

Arrivée (autorisée / bloquée)

Empêche les voyageurs d'arriver ce jour précis (les séjours en cours peuvent continuer). Pratique pour caler les arrivées sur des jours fixes (ex. arrivées le samedi uniquement).

Départ (autorisé / bloqué)

Empêche les voyageurs de partir ce jour précis. Même logique que l'arrivée, pour organiser vos rotations et ménages.

Chaque réglage peut être appliqué à une période (date de début à date de fin) et filtré par jours de la semaine. Exemple : « +30 % sur les vendredis et samedis de juillet ».

4. Les nuits orphelines

Une nuit orpheline est une nuit libre qu'aucun voyageur ne peut réserver à cause de vos contraintes. Exemple typique : il vous reste une seule nuit libre entre deux réservations, mais votre minimum de séjour est de 2 nuits — personne ne peut donc la réserver. Cette nuit risque de rester vide.

HôteSmart les détecte automatiquement et les fait ressortir en orange avec une alerte ⚠️ dans le calendrier.

Correction automatique

Quand activée, HôteSmart ajuste les contraintes (réduit le minimum de nuits, lève les blocages d'arrivée/départ) sur les nuits orphelines pour les rendre réservables. Ce réglage est conservé et suit votre bien sur tous vos appareils.

Prix spécial nuits orphelines

Appliquez un tarif attractif (fixe, ou en plus/moins, ou en pourcentage) pour inciter à réserver ces nuits difficiles. Une remise de dernière minute sur ces nuits améliore votre taux d'occupation.

5. Tutoriels pas à pas

💶 Augmenter mes prix le week-end
  1. Ouvrez le calendrier de votre bien
  2. Ouvrez le panneau de configuration (clic ou toucher sur une date)
  3. Dans Prix de base, choisissez la période concernée
  4. Sélectionnez uniquement Ven et Sam dans le filtre des jours
  5. Entrez le montant (ou utilisez +/− pour ajuster)
  6. Enregistrez : c'est publié automatiquement
🔒 Bloquer une période (usage personnel)
  1. Ouvrez le panneau de configuration
  2. Réglez la période de début et de fin
  3. Dans Disponibilité, choisissez Fermé
  4. Enregistrez : les dates passent en rouge, plus aucune réservation possible
🌙 Imposer un minimum de 3 nuits en haute saison
  1. Ouvrez le panneau, réglez la période (ex. juillet-août)
  2. Dans Minimum de nuits à l'arrivée, entrez 3
  3. Enregistrez
⚠️ Récupérer mes nuits orphelines
  1. Repérez les dates en orange dans le calendrier
  2. Touchez-en une : la rubrique Nuits orphelines s'ouvre
  3. Activez Correction automatique
  4. (Optionnel) Activez un prix spécial attractif
  5. Enregistrez : ces nuits redeviennent réservables

6. Questions fréquentes

Où sont enregistrées mes modifications ?

Vos tarifs et restrictions sont enregistrés de façon permanente et publiés sur vos plateformes connectées. HôteSmart reste la référence de votre historique.

Que se passe-t-il pour une date que je n'ai pas configurée ?

Elle utilise automatiquement le prix de base de votre bien. Vous n'avez à configurer que les exceptions.

Mes réglages sont-ils les mêmes sur mobile et ordinateur ?

Oui. Les tarifs, restrictions et la correction automatique des nuits orphelines sont partagés. Seul l'affichage diffère (tableau sur ordinateur, calendrier mensuel sur mobile).
