🏨 Beds24
Vérification…
1
Connectez-vous à Beds24 et allez dans Paramètres → Marketplace → API
2
Cliquez sur Generate invite code. Cochez READ + WRITE sur bookings et properties. Sélectionnez All properties.
3
Copiez l'invite code généré et collez-le ci-dessous. Attention : l'invite code est à usage unique.
4
Cliquez sur Connecter — HôteSmart récupère et sauvegarde automatiquement le token.
Le token Beds24 expire après 24h. HôteSmart renouvelle automatiquement via le refresh token. Si la connexion échoue, générez un nouvel invite code.
